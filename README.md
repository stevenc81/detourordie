detourordie
===========

Report police checkpoints on streets


##Components
* API service
* Mobile web
* Streaming service


##Installation
* Install Express
* npm insatll in /, /mobileweb, and /asms

##Running
* node app.js in the above directories

##For Dev
* grunt watch in /mobileweb to compile jade and less files in real time
* Install redis
* Install MongoDB
